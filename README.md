# todo-list

A simple JavaScript-based Todo List app with date reminders and item management.

## Project Structure

- `todo-list/todo-list.html` - main HTML page
- `todo-list/todo-list.css` - styles
- `todo-list/todo-list.js` - interaction logic

## Features

- Add todo items with a due date
- Delete individual items
- Clear all items
- Basic DOM rendering and state management using an array

## How to run locally

Option 1 (open directly):
1. Open `todo-list/todo-list.html` in your browser.

Option 2 (local server):
1. Open terminal in the `todo-list` folder.
2. Run `python -m http.server 5500` (or `py -m http.server 5500`).
3. Open `http://localhost:5500/todo-list.html`.

## Styling

- Grid layout for input and header elements
- Responsive small layout with clear action buttons

## Notes

- No backend; state is in-memory and clears on refresh.
- Can be extended with localStorage or persistent backend.
